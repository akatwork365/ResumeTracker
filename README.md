A simple and efficient Resume Tracker Web Application built using HTML, CSS, Flask, and MySQL.
This project helps users manage, track, and organize job applications in one place.

🚀 Features
🔐 User Authentication (Login & Signup)
📂 Add and Manage Resume Applications
🏢 Store Company Details
📅 Track Application Status
✏️ Update or Delete Entries
📊 Clean Dashboard UI
💾 MySQL Database Integration
⚡ Flask Backend Routing
🛠️ Tech Stack
Frontend
HTML5
CSS3
JavaScript
Backend
Flask (Python)

⚙️ Installation
1️⃣ Clone the Repository
git clone https://github.com/akatwork365/ResumeTracker.git
cd resume-tracker
2️⃣ Create Virtual Environment
python -m venv venv
3️⃣ Activate Virtual Environment
Windows
venv\Scripts\activate
Mac/Linux
source venv/bin/activate
4️⃣ Install Dependencies
pip install -r requirements.txt
5️⃣ Setup MySQL Database
Create a database in MySQL
Import database.sql
CREATE DATABASE resume_tracker;
6️⃣ Run Flask Application
python app.py
